# 프로젝트 비전 및 전략

## 목표

- **월간 방문자**: 1,000명 (4개월 내)
- **포스팅 빈도**: 주 3-4개 (자동화)
- **비용**: 완전 무료
- **차별화**: 데이터 큐레이션 + Moltbook 커뮤니티 검증

## 핵심 원칙

```
AI 생성 콘텐츠만으로는 실패
↓
데이터 집계 + 커뮤니티 피드백 = 성공
```

## 불편한 진실

### AI 단독 운영의 한계

**절대 이길 수 없는 것들**
- ❌ 직접 경험 블로그 (실제 방문 후기)
- ❌ 실시간 정보 (오늘 영업하는지, 가격 변동)
- ❌ 감성적 공감 (여행의 설렘, 실망)

**AI가 이길 수 있는 영역**
- ✅ 정보 집계/큐레이션 ("서울 전시회 일정 총정리")
- ✅ 데이터 분석 ("2026년 인기 여행지 TOP 20")
- ✅ 가이드/팁 ("제주도 렌터카 가격 비교")
- ✅ 역사/문화 해설 ("경복궁 건축 양식 분석")

## 블로그 포지셔닝

```
기존: "내가 다녀온 여행 후기" ❌
개선: "AI 큐레이터가 정리한 여행 & 문화예술 데이터베이스" ✅
```

**슬로건**: "1,000개 블로그를 읽고 정리했습니다"

## 콘텐츠 전략

### 콘텐츠 유형별 성공률

| 유형 | 성공률 | 예시 |
|------|--------|------|
| 데이터 집계형 | 90% | "제주도 렌터카 가격 비교 2026" |
| 일정 큐레이션형 | 80% | "서울 1박2일 문화예술 코스" |
| 분석/인사이트형 | 70% | "2026년 2월 가장 핫한 여행지 TOP 10" |

### SEO 전략: AI 감별 우회

```markdown
❌ AI 티나는 글:
"제주도는 아름다운 자연경관으로 유명합니다."

✅ 데이터 기반 큐레이션:
"네이버 검색량 분석 결과, 2월 제주 여행객의 68%가
'한라산 눈꽃' 키워드로 검색합니다. (출처: 네이버 데이터랩)

| 명소 | 2월 검색 비중 | 평균 체류시간 |
|------|--------------|--------------|
| 한라산 | 34% | 4.2시간 |
```

## 4개월 로드맵

### 1개월: 인프라 구축 + 초기 콘텐츠 (목표: 일 50명)

- Week 1: 시스템 구축
  - Hugo 블로그 세팅
  - GitHub Actions 배포 자동화
  - Moltbook 등록 및 첫 포스트

- Week 2: 데이터 소스 연동
  - 한국관광공사 API 연동
  - 문화포털 API 연동
  - Unsplash 이미지 연동

- Week 3-4: 콘텐츠 생산 (50개)
  - 일 2개 포스트 자동 생성
  - Moltbook 매일 공유
  - 피드백 수집 시작

### 2개월: 롱테일 확장 + 피드백 반영 (목표: 일 300명)

- 콘텐츠 대량 생산 (추가 70개, 총 120개)
- 롱테일 키워드 조합: "지역 + 시설 + 정보"
- Moltbook 피드백 활용:
  - 주간 피드백 분석
  - 요청사항 우선 생산
  - 개선 사항 즉시 반영

### 3개월: 시즌 키워드 + 커뮤니티 협업 (목표: 일 700명)

- 시즌 콘텐츠 (40개)
  - "벚꽃 명소 2026"
  - "봄 축제 일정"
- Moltbook 커뮤니티 협업
  - 다른 agent와 협업 포스트
  - 커뮤니티 투표 활용

### 4개월: 최적화 + 확장 (목표: 일 1,200명)

- 기존 콘텐츠 업데이트
- Moltbook 영향력 확대
  - 월간 "베스트 큐레이션" 시리즈
  - 커뮤니티 이벤트 주최

## Moltbook 피드백 루프

```
블로그 포스트 발행
↓
Moltbook에 자동 공유
↓
AI 커뮤니티 피드백 수집
↓
인기 주제 / 요청사항 / 개선점 분석
↓
콘텐츠 전략 자동 조정
↓
다음 포스트에 반영
```

## 성공 지표

### 블로그 성과
- 일일 방문자
- 검색 노출 순위
- 인덱싱된 페이지 수

### Moltbook 성과
- Karma 점수
- 평균 upvotes/post
- 댓글 engagement rate
- 팔로워 수

## 일일 워크플로우

```bash
# 아침 (09:00) - 콘텐츠 생성
npm run collect -- -k "키워드"
npm run new -- -t "키워드" --type travel
npm run review
npm run publish

# 낮 (12:00) - Heartbeat
npm run moltbook:heartbeat

# 저녁 (18:00) - 피드백 분석
npm run moltbook:feedback
npm run moltbook:analyze
```
